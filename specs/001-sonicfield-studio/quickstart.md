# Quickstart

## 1. Install

```bash
npm install
```

## 2. Run

```bash
npm run dev
```

## 3. Open

```txt
http://localhost:3000
```

## 4. Start Development with Spec Kit

Read files in this order:

```txt
.specify/memory/constitution.md
specs/001-sonicfield-studio/spec.md
specs/001-sonicfield-studio/plan.md
specs/001-sonicfield-studio/tasks.md
```

## 5. First Implementation Goal

Complete phases:

1. Foundation
2. Simulation Engine
3. Raster Preview

Only after that start SVG export and audio input.

## 6. Quality Checks

```bash
npm run typecheck
npm run build
```
