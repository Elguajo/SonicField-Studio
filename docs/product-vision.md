# Product Vision

SonicField Studio should become a simple but powerful generative design tool.

## Positioning

A tool for creating visual assets from sound, rhythm, frequency, and mathematical wave systems.

## Use Cases

- Poster backgrounds
- Album cover graphics
- Social media visuals
- Motion design frames
- Brand pattern systems
- Abstract visual identities
- Print-ready vector patterns

## Product Personality

- precise;
- experimental;
- visual;
- professional;
- designer-friendly.

## What It Should Not Become

- a generic music player;
- a DAW;
- a complex node editor;
- a video editor;
- an AI image generator;
- a template marketplace in MVP.

## Future Commercial Direction

Possible monetization later:

- Pro export sizes
- Preset packs
- Video export
- Brand pattern kits
- Figma/Illustrator plugin
- Paid desktop version
